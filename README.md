# Travellers - A Mapify Clone
First big project in group during my studies

# Date
11th November 2020

# How to Setup
- cd travellers
- npm install
- npm run dev

# Languages used in Front
- HTML 5
- CSS 3
- JavaScript 

# Framework used in Front
- Bootstrap via cdn
- Vuejs Cli
- Vue Router
- VueX
